Brightway2 regionalized LCA
===========================

This package provides a regionalized LCA calculations for the `Brightway2 life cycle assessment framework <http://brightwaylca.org>`_. `Online documentation <https://brightway2-regional.readthedocs.org/en/latest/>`_ is available, and the source code is hosted on `Bitucket <https://bitbucket.org/cmutel/brightway2-regional>`_.
